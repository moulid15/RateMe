About Us

This social media platform started out as a Hackathon project created by a group of college students.

Goal

The goal of this project is to create a social media platform that allows people to rate each other. The challenges we face during this project helped us work as a team to problem solve and it helped us learn new tools to build our project. We hope to create something enjoyable for the users and for us.
